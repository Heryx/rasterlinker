GeoSurvey Studio

GeoSurvey Studio is a QGIS plugin for geophysical/time-slice workflows.

Core Features

  • Grouped raster management for time-slice datasets.
  • Grid drawing and orientation tools for survey planning.
  • 2D/3D trace workflow with dedicated drawing panel.
  • Project manager with catalog and import/export utilities.

Main Files

  • geosurvey_studio.py
  • geosurvey_studio_dialog.py
  • geosurvey_studio_dialog_base.ui
  • metadata.txt
  • resources.qrc / resources.py

Development Quick Start

  1. Copy the plugin folder into your QGIS profile plugin directory.
  2. Rebuild Qt resources when icon/resource paths change:
    python-qgis -m PyQt5.pyrcc_main -o resources.py resources.qrc
  3. Reload the plugin in QGIS and test the main workflows.
  4. Run repository tests when supported by your environment.
